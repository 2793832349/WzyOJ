## 💡 快速参考

### 常用命令
```bash
# 快速启动
./start-videocourse.sh

# 查看 API 状态
curl -H "Authorization: Bearer YOUR_TOKEN" \
  http://localhost:8000/api/course-chapters/

# 查看 Celery 状态
celery -A oj_backend inspect active

# 查看日志
tail -f /tmp/celery.log
```

### 关键文件位置
```
前端页面：
  frontend-naive/src/pages/videocourse/
  ├── index.vue (列表)
  ├── _id.vue (详情)
  └── edit.vue (编辑)

前端组件：
  frontend-naive/src/components/
  ├── VideoPlayer.vue (播放器)
  └── VideoCourseManager.vue (管理)

后端模块：
  backend/oj_course/
  ├── models.py (数据模型)
  ├── views.py (API 视图)
  ├── video_processor.py (视频处理)
  └── tasks.py (异步任务)

路由配置：
  frontend-naive/src/router/index.js (第 307-355 行)
```

### 关键地址
```
录播课列表：http://localhost:5173/videocourse/
录播课详情：http://localhost:5173/videocourse/:id/
创建课程：http://localhost:5173/videocourse/create/
编辑课程：http://localhost:5173/videocourse/:id/edit/
API 端点：http://localhost:8000/api/course-chapters/
Admin 后台：http://localhost:8000/admin/
```
